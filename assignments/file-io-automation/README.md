# 📘 Assignment: File I/O and Automation

## 🎯 Objective

Learn how to read from and write to files in Python, then build a small automation script that processes data and generates a report.

## 📝 Tasks

### 🛠️ Read and parse data from a file

#### Description
Create a Python script that loads a text or CSV file, parses its contents, and summarizes the data.

#### Requirements
Completed program should:
- Open and read file contents using Python's built-in file I/O.
- Parse each line or record into meaningful values.
- Handle missing or malformed lines gracefully.
- Use a function to encapsulate file reading logic.

### 🛠️ Write results to a new file

#### Description
Write processed data or summary results back to a new output file.

#### Requirements
Completed program should:
- Create an output file with a summary or transformed data.
- Include headers or clear labels in the output.
- Overwrite the file if it already exists.
- Use a separate function for file writing.

### 🛠️ Automate the workflow

#### Description
Build a main script that runs the file processing end to end and can be executed from the command line.

#### Requirements
Completed program should:
- Accept an input filename and output filename.
- Read the input file, process the data, and write results to the output file.
- Print a short completion message when done.
- Include simple error handling for missing input files.
