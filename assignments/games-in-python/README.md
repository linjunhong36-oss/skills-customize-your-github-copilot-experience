
# 📘 Assignment: Games in Python

## 🎯 Objective

Build a Hangman word-guessing game using Python control flow, string handling, and user interaction.

## 📝 Tasks

### 🛠️ Create the Hangman Game

#### Description

Implement a Hangman game that randomly chooses a secret word and lets the player guess letters until the word is solved or the player runs out of attempts.

#### Requirements
Completed program should:

- Randomly select a word from a predefined list.
- Accept letter guesses from the player using `input()`.
- Reveal correctly guessed letters in the word and hide remaining letters with underscores.
- Track and display the number of incorrect guesses remaining.
- End the game with a clear win or lose message.
- Prevent the player from guessing the same letter more than once.

### 🛠️ Add User Feedback and Game Flow

#### Description

Improve the game experience by showing current progress and handling invalid input.

#### Requirements
Completed program should:

- Display the current word progress after each guess.
- Show letters that have already been guessed.
- Handle invalid input such as empty responses or non-letter guesses.
- Allow the player to play again or exit after the game ends.
